Rail Structure

Position Item Type Contains
Top Logo + Panel toggle Fixed —
1 💬 Chat Standalone Direct → /
2 📊 Dashboard Standalone Direct → /dashboard
── separator ──
3 📄 Documents Group (flyout) Documents, Create, Compare
4 ⚖️ Legal Tools Group (flyout) Clause Library, Compliance, Analysis, Obligations
5 🏛️ Governance Group (flyout) Approvals, Regulatory, Integrations
── spacer (flex-1) ──
Bottom ⚙️ Settings Standalone Direct → /settings
Bottom Avatar Fixed User info

Interaction

  • Chat / Dashboard / Settings: Click → navigate directly (1-click)
  • Group icons: Click → flyout popover appears to the right with items listed (icon + label)
  • Flyout item: Click → navigate to page, flyout closes
  • Active indicator: If any child page in a group is active, the group icon gets the active style (purple bg + left gradient border)
  • Tooltip: Hover on group icon shows group name (e.g., "Documents")
