Flat fill is cleaner, loads faster, and is the 2026 enterprise standard. The gradient on the chat send button (InputActionBar) also becomes solid.

Glass Treatment

Where Glass Lives Now vs Proposed
Element Current Proposed
ThreadPanel (sidebar) ✅ glass + dot-grid Solid surface (#111113), no blur
10 page sidebar panels ✅ glass Solid surface (#111113), no blur
Chat input card ✅ glass Solid card surface (#18181B)
Tooltips ✅ glass + blur ✅ Keep glass — transient overlay
Popovers / dropdowns Default ✅ Add glass — transient overlay
Mobile panel overlay ✅ backdrop-blur ✅ Keep — transient overlay

Rule: glass = transient only (tooltips, popovers, mobile overlays). Persistent panels get solid surfaces. This reduces GPU load and matches 2026 convention.
