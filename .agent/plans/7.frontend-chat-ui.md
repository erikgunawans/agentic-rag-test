# Plan 7: Frontend Chat UI
🔴 **Complex** - Full chat interface with SSE streaming

## Goal
Thread sidebar + message view + streaming chat input.

## Tasks

### 7.1 frontend/src/components/chat/ThreadList.tsx
- List of threads from `GET /threads`
- "New Thread" button → `POST /threads` then navigate to it
- Active thread highlighted
- Click → navigate to thread
- Delete button (trash icon) per thread
- Loads on mount, refreshes on thread create/delete

### 7.2 frontend/src/components/chat/MessageView.tsx
- List of messages for active thread
- Auto-scrolls to bottom on new message
- Shows user messages right-aligned, assistant left-aligned
- Shows streaming message as it arrives (via `StreamingMessage`)
- Empty state: "Send a message to start"

### 7.3 frontend/src/components/chat/StreamingMessage.tsx
- Accepts `content: string` prop (grows as tokens arrive)
- Renders markdown content (use `react-markdown` if available, else plain text)
- Shows blinking cursor while streaming

### 7.4 frontend/src/components/chat/MessageInput.tsx
- Textarea with send button
- Enter to send (Shift+Enter for newline)
- Disabled while streaming
- Clears on send

### 7.5 frontend/src/pages/ChatPage.tsx
- Layout: sidebar (ThreadList) + main area (MessageView + MessageInput)
- State: `threads`, `activeThreadId`, `messages`, `isStreaming`
- `sendMessage(content)`:
  1. Add optimistic user message to UI
  2. `POST /chat/stream` with `fetch` + `ReadableStream`
  3. Read SSE events, parse `data: {...}` lines
  4. Append deltas to streaming message
  5. On `done: true`, replace streaming message with final
  6. Update thread `updated_at` in sidebar
- Sign out button in header/sidebar

### 7.6 SSE consumption pattern
```typescript
const response = await apiFetch('/chat/stream', {
  method: 'POST',
  body: JSON.stringify({ thread_id, message }),
});
const reader = response.body!.getReader();
const decoder = new TextDecoder();
let buffer = '';
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  buffer += decoder.decode(value, { stream: true });
  const lines = buffer.split('\n');
  buffer = lines.pop()!;
  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const event = JSON.parse(line.slice(6));
      // handle delta or done
    }
  }
}
```
