# Plan 1: Project Setup
✅ **Simple** - Single-pass executable, low risk

## Goal
Scaffold monorepo structure: Vite React frontend + FastAPI Python backend, env var config, dev tooling.

## Tasks

### 1.1 Create directory structure
- `frontend/` — Vite + React + TypeScript + Tailwind + shadcn/ui
- `backend/` — FastAPI + Python venv
- `supabase/migrations/` — SQL migration files

### 1.2 Frontend scaffold
- Run `npm create vite@latest frontend -- --template react-ts`
- Install: `tailwindcss`, `@tailwindcss/vite`, `@supabase/supabase-js`, `lucide-react`
- Install shadcn/ui: `npx shadcn@latest init`
- shadcn components: `button`, `input`, `scroll-area`, `separator`, `sheet`, `skeleton`
- Configure `tailwind.config.ts` and `vite.config.ts`

### 1.3 Backend scaffold
- Create `backend/venv/` with `python3 -m venv venv`
- Create `backend/requirements.txt`
- Create `backend/run.sh`
- Create `backend/app/__init__.py`

### 1.4 Environment config
- Create `.env.example` at project root listing all required vars
- Create `frontend/.env.example` for VITE_ vars
- Create `backend/.env.example` for backend vars
- Add `.env` and `backend/venv/` to `.gitignore`

## Validation
- `ls frontend/src` → shows React scaffold
- `ls backend/app` → shows Python module
- `cat .env.example` → shows all required env vars
