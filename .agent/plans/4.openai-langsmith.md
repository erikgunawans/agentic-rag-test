# Plan 4: OpenAI + LangSmith Services
⚠️ **Medium** - OpenAI Responses API with LangSmith tracing

## Goal
Create OpenAI Responses API service and LangSmith observability wrapper.

## Tasks

### 4.1 backend/app/services/langsmith_service.py
- `configure_langsmith()` → sets LANGCHAIN_TRACING_V2, LANGCHAIN_PROJECT env vars
- Called at app startup if LANGSMITH_API_KEY is set
- No-op if not configured

### 4.2 backend/app/services/openai_service.py
- `OpenAIService` class initialized with `openai.AsyncOpenAI(api_key=...)`
- `create_thread()` → calls `client.beta.threads.create()` → returns OpenAI thread ID
  - Decorated with `@traceable` from langsmith
- `stream_response(thread_id, last_response_id, message, vector_store_id)` → async generator
  - Uses `client.responses.create(stream=True, ...)`
  - `model="gpt-4o-mini"`
  - `input=[{"role": "user", "content": message}]`
  - `previous_response_id=last_response_id` if set
  - `tools=[{"type": "file_search", "vector_store_ids": [vector_store_id]}]` if vector_store_id set
  - Yields text deltas as SSE events: `data: {"delta": "...", "done": false}\n\n`
  - Final yield: `data: {"delta": "", "done": true, "response_id": "..."}\n\n`
  - Decorated with `@traceable`

## Key API Pattern
```python
# OpenAI Responses API (NOT chat.completions)
response = await client.responses.create(
    model="gpt-4o-mini",
    input=[{"role": "user", "content": message}],
    previous_response_id=last_response_id,  # thread continuity
    tools=[{"type": "file_search", "vector_store_ids": [vs_id]}],
    stream=True
)
async for event in response:
    if event.type == "response.output_text.delta":
        yield event.delta
```

## Validation
- Service instantiates without error
- `@traceable` decorator applied (visible in LangSmith after real call)
