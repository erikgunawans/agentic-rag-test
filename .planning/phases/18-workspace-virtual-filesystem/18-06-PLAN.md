<objective>
Add the `workspace_updated` SSE event to the chat-loop event taxonomy. Three emission points:

1. **Direct tool calls (`write_file`, `edit_file`)** — emitted from `chat.py` immediately after the corresponding `tool_result` event, when the tool returned `{"ok": True, ...}`. 3. **No emission for `read_file` / `list_files`** (read-only).

Frontend consumption is plan 18-07.

Purpose: WS-10.
